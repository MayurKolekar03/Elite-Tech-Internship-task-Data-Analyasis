🌍 Big Data Analysis – Global Earthquake Dataset
📌 Project Overview
This project performs scalable big data analysis on global earthquake data obtained from the United States Geological Survey (USGS).

🛠 Tools Used
Python
PySpark
Google Colab
Matplotlib

📊 Dataset Information
Source: USGS Earthquake Catalog
Records: Large-scale global seismic activity data
Attributes: Time, Latitude, Longitude, Depth, Magnitude, Place,etc.

🔍 Key Analysis Performed
Top earthquake-prone locations
Average magnitude by year
Depth vs magnitude relationship
Highest magnitude recorde

📈 Key Insights
Earthquakes are geographically concentrated.
Magnitude trends are relatively stable yearly.
Tsunami-triggering earthquakes are rare.

✅ Conclusion
The project demonstrates big data scalability using PySpark and provides meaningful geospatial insights from large earthquake datasets.
