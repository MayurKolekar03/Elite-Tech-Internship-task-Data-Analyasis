# Task 4 – NLP Text Analysis on Earthquake Data

## Objective

Apply Natural Language Processing (NLP) techniques on earthquake location text to classify events into impact categories.

## Dataset

Earthquake dataset containing:

* Latitude
* Longitude
* Depth
* Magnitude
* Place (text data)

## NLP Techniques Used

* Text Cleaning
* Stopword Removal
* TF-IDF Vectorization

## Machine Learning Model

Logistic Regression

## Model Performance

Accuracy: **91%**

## Insights

The model successfully captured textual patterns in earthquake location descriptions and classified events into High Impact and Low Impact categories.

## Tools Used

* Python
* Pandas
* Scikit-learn
* NLTK
